# Security Policy

## Supported Versions

当前仓库处于快速迭代阶段，默认仅维护最新主分支与最新发布版本。

| Version | Supported |
|---------|-----------|
| latest  | yes       |
| older   | no        |

## Reporting a Vulnerability

请不要在公开 Issue 中直接披露安全细节。  
建议通过以下方式私下反馈：

1. 在 GitHub 创建私密安全报告（Security Advisories）。
2. 或在 Issue 中仅描述“存在安全问题”，不附 PoC 和敏感细节，并请求维护者私下联系。

反馈建议包含：

- 影响范围（模块、接口、版本）
- 复现步骤
- 预期影响
- 临时缓解建议（如有）

维护者会在确认后尽快评估并给出修复计划。
