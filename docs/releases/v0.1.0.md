# 🎉 v0.1.0 首版发布说明

首个开源版本已完成发布，目标是提供一个可直接部署、可长期运行、兼容 OneBot v11/v12 的 LLM 聊天插件基线。

---

## ✨ 版本亮点

- OneBot v11 / v12 双协议接入（best-effort + 自动降级）
- 完整的对话能力：上下文记忆、搜索工具、图片理解、主动发言
- WSL2 使用指南（可选）
- 可复用的启动脚本：`start.sh`（Linux/WSL）与 `start.ps1`（Windows）
- 发布流水线已就位：Tag 触发 GitHub Release 自动打包

---

## 🧩 兼容性与降级策略

本版本重点强调“可运行优先”：

- 适配器缺失可选能力时（如 Forward / Quote / 历史接口），自动降级而不是中断主流程
- v12 的 mention / image 差异采用兼容解析，不要求实现方行为完全一致
- Prompt 配置异常场景采用防御性降级，避免因自定义配置导致启动失败

---

## 🚀 开箱即用

最小必填配置：

- `MIKA_API_KEY`（或 `MIKA_API_KEY_LIST`）
- `MIKA_MASTER_ID`

快速启动：

```bash
cp .env.example .env
./start.sh
```

OneBot 反向 WS 地址：

- v11: `ws://<HOST>:<PORT>/onebot/v11/ws`
- v12: `ws://<HOST>:<PORT>/onebot/v12/ws`
